SuppApp
=====================
An interactive mobile app for iOS aimed to people find supply locations.

Built using React Native, with Flux.

## Clone this repo

> git clone https://github.com/leonardocouy/suppapp.git

## How to run? ##

* `npm install`
* `react-native run-ios`

## How to run? (Developer) ##

* `npm install --dev`
* `react-native run-ios`
